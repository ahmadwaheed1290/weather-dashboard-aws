# Live Weather Dashboard — AWS Lambda + OpenWeatherMap

A serverless weather dashboard that fetches real-time weather data for any city in the world — temperature, humidity, wind speed, and conditions delivered in under two seconds.

## Live Demo
https://d2uko7csallknv.cloudfront.net/weather.html

## Tech Stack
- AWS Lambda
- API Gateway
- OpenWeatherMap API
- Python
- Amazon S3 + CloudFront

## How It Works
1. User enters a city name
2. API Gateway receives the POST request
3. Lambda calls the OpenWeatherMap API
4. Weather data is processed and formatted
5. Results displayed instantly on the dashboard

## Features
- Real-time temperature, humidity, and wind speed
- Auto-detected weather conditions
- Global city support
- No signup or account required
